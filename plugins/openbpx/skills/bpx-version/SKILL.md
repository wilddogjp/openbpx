# version

> **PREREQUISITE:** Read [bpx-shared](../bpx-shared/SKILL.md).

## Usage

```bash
bpx version
```

## Behavior

- Prints the CLI semantic version string and exits with code 0.
- No package file is parsed.

## High-Signal Examples

```bash
bpx version
```
